## 1. OVERVIEW

### 1.1 Purpose

This specification defines a **generic GIS polygon capture component** for Joget DX8. The plugin provides pure geometry capture functionality that can be used for any polygon-based data: land parcels, farm boundaries, irrigation zones, protected areas, administrative boundaries, building footprints, etc.

### 1.2 Design Philosophy

| Principle | Description |
|-----------|-------------|
| **Generic** | No business-domain assumptions â€” pure geometry operations |
| **Composable** | Works with standard Joget form fields for business attributes |
| **Configurable** | All behavior controlled via plugin properties |
| **Responsive** | Draw Mode optimized for desktop, Walk Mode optimized for mobile |
| **Offline-Capable** | Walk Mode works without internet (GPS is device-local) |

### 7.3 Form Load Behavior

When form loads with existing data:
1. Read GeoJSON from `fieldId` field
2. Parse and display polygon on map
3. Calculate and display metrics
4. If edit mode â†’ show [Edit] button

### 7.4 Form Submit Behavior

When user saves:
1. GeoJSON written to `fieldId` field
2. Calculated values written to mapped fields
3. Standard Joget form submission handles all fields

## 10. SUMMARY

### 10.1 Key Design Decisions

| Decision | Rationale |
|----------|-----------|
| **Generic plugin** | Reusable for any polygon capture use case |
| **Separate from business fields** | Clean separation, Joget-native approach |
| **Mode selection** | Different tools for different contexts |
| **Real-time feedback** | Users verify as they draw/walk |
| **Configurable validation** | Different rules for different use cases |
| **Optional overlap check** | Enable only when needed |

### 10.2 Plugin Outputs

| Output | Type | Description |
|--------|------|-------------|
| `geometry` | GeoJSON | The captured polygon |
| `area` | Decimal | Area in hectares |
| `perimeter` | Decimal | Perimeter in meters |
| `centroid` | GeoJSON | Center point |
| `vertexCount` | Integer | Number of vertices |

### 10.3 Usage Examples

**Land Parcel Registration:**
- GIS plugin for boundary
- Standard fields for: parcel name, land use, tenure type, farmer reference

**Protected Area Management:**
- GIS plugin for boundary
- Standard fields for: area name, protection level, restrictions

**Farm Boundary Mapping:**
- GIS plugin for boundary
- Standard fields for: farm name, owner, total area, primary crop

**Irrigation Zone Definition:**
- GIS plugin for boundary
- Standard fields for: zone name, water source, schedule
